# Avinash Jha - Full-Stack & Mobile App Developer Portfolio

A modern, responsive portfolio website showcasing professional work and skills in full-stack and mobile app development.

## 🚀 Features

- **Modern Design**: Glassmorphism effects, gradient animations, and 3D transforms
